# Proyecto de Navegación - Optimización de Rutas

## Descripción
Sistema para calcular rutas óptimas en un grafo de calles usando algoritmos Dijkstra y A*.  
Incluye backend PHP con PostgreSQL y frontend interactivo con mapa Leaflet.

## Tecnologías
- PHP 7.4+
- PostgreSQL 13+
- HTML5, CSS3, JavaScript
- Leaflet (mapas)
- PlantUML (diagramas)
